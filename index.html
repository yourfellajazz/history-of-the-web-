History of the web

Brief introduction of the beginning stages of the web.

Tim Berners-Lee

Introduced the beginning stages of the World Wide Web when he was working at CERN.

First web browser

The first web browser was introduced by Berners-Lee. It was called "WorldWideWeb". Along with that, he also created the first web server and web page.

First widely used web browser

The first browser that was used extensively more was called Mosaic. It was later renamed to Nexus in order to avoid confusion with the actual World Wide Web.
